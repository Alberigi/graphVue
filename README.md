Web application developed with authentication system and user CRUD.Front-end in VueJs using VueRouter and Vuex, consuming a GraphQl API in NodeJs with a SQL database (Mysql).

## Run project

### Run Front-end

  - `npm install`
  - `npm run build`
  - `npm run serve`
  
### Run Back-end

- `npm install`
- `npm run build`
- `knex migrate:latest`
- `npm run start`
